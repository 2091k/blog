网页下载安装好打印插件后，网页打印还是提示未安装打印插件。

原因：

现代

Chrome/Edge 安全策略：外网域名 http://srm.rjetech.com:8000 （公网 IP / 域名）去访问本地localhost属于私有地址空间跨域，非 HTTPS 环境直接拦截本地 CLodop 脚本。

# 解决办法

# Chrome 浏览器操作

地址栏输入：

```
chrome://flags/#block-insecure-private-network-requests
```

找到Block insecure private network requests→ 改为Disabled

再输入：

```
chrome://flags/#local-network-access-check
```

Local Network Access Checks→ 改为Disabled

页面底部点Relaunch重启浏览器

底部点Relaunch重启浏览器，刷新打印页面即可
